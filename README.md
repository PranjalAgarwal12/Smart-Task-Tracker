# Smart Task Tracker (MERN)

A MERN-based full-stack task management application with secure JWT authentication and user-specific task handling.

## Features
- User authentication (Register / Login)
- JWT-based authorization
- Create, update, delete tasks
- Task status tracking
- User-specific task storage
- Protected backend APIs

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT

## Project Structure
- frontend: React user interface
- backend: REST APIs, authentication, database models
